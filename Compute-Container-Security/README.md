# Module 03 — Compute & Container Security

VMs, serverless functions, and Kubernetes. The compute plane is where attackers *live* after they get access — so it dominates Detection Engineering. Covers escape primitives, supply chain, runtime hardening, and admission control.

## Learning objectives

- Harden VMs and function runtimes per cloud.
- Explain container escape classes and Kubernetes attack surfaces.
- Implement admission control, pod security, and image supply chain.
- Detect malicious compute activity from logs/telemetry.
- Distinguish AW-hosted Kubernetes from self-managed control plane responsibilities.
