# Module 04 — Storage & Data Security

Object storage, block volumes, managed DB snapshots, backups. Public exposure here = canonical cloud breach headline. This module covers encryption, exposure control, ransomware against cloud backups, and copy-protected recovery.

## Learning objectives

- Configure at-rest encryption with customer-managed keys per cloud.
- Detect and prevent public exposure of buckets/containers/projects.
- Model the "ransomware kills your cloud backups too" reality.
- Implement immutability / object-lock / WORM for recovery.
- Compare managed DB encryption, TDE, and network exposure per cloud.
