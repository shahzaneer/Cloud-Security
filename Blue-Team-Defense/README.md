# Module 10 — Blue Team: Cloud Defense

The defense counterpart to Module 09. Harden baselines, build preventive guardrails as code, reduce blast radius, deploy deception (honey tokens), automate auto-response (isolate/quarantine), and translate the red-team chain into controls.

## Learning objectives

- Establish landing-zone baselines (Org / Management Group / Folder) with preventive guardrails.
- Reduce blast radius through account isolation, partitioning, break-glass.
- Deploy canary/honey tokens cross-cloud.
- Build per-incident auto-response playbooks that isolate a compromised role/instance.
- Map Module-09 offense techniques 1-to-1 to defensive controls.
